Project Title: Udemy Paid Course Analysis
Objective:
The project's goal is to learn about Udemy user preferences in order to help the Udemy platform filter course characteristics that Udemy users are most interested in. Using descriptive and inferential analysis to determine the relationship between each data feature and gain business insight from the results
Data source: https://www.kaggle.com/datasets/andrewmvd/udemy-courses
Data visualization link : https://lookerstudio.google.com/reporting/7da6302f-a248-4361-b5df-a9dfced3d219
